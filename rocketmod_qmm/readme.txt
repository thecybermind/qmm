Win32 Installation:
	1. Install QMM
	2. Make a qmmaddons/rocketmod/dlls directory inside your mod directory and place rocketmod_qmm.dll here
	3. Add "qmmaddons/rocketmod/dlls/rocketmod_qmm.dll" as an entry in the plugins list in qmm.ini

Linux Installation:
	1. Install QMM
	2. Make a qmmaddons/rocketmod/dlls directory inside your mod directory and place rocketmod_qmm.so here
	3. Add "qmmaddons/rocketmod/dlls/rocketmod_qmm.so" as an entry in the plugins list in qmm.ini
